1. download aplikasi CLIPS 
nek 32bit -> https://sourceforge.net/projects/clipsrules/files/CLIPS/6.31/clips_631_windows_32_bit_installer.msi/download
nek 64bit -> https://sourceforge.net/projects/clipsrules/files/CLIPS/6.31/clips_631_windows_64_bit_installer.msi/download
2. install trus foldernya serah
3. extract file zip di folder CLIPS yang udh dibikin tadi (ga jg gpp sih tp biar enak)
4. di folder CLIPS pencet CLIPSIDE64 atau CLIPSIDE32 ya sesuai sama brp bit laptop anda
5. nek udah kebuka pencet ctrl + l atau ga pencet file -> load
6. cari file mbti dari zip yg dah diekstrak tadi
7. trus yaudah kalo sukses bakalan keload filenya
8. habis tu tinggal ketik (run)
9. dah deh programe jalan tinggal isi
